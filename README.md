# Team 20 – AI Innovation Student Website

A modern student-focused website developed for our Quinnipiac web design and AI innovation assignment.

This project showcases how AI-assisted development, modern UI/UX principles, and collaborative Git workflows can be used to build a professional and responsive website.

---

## Live Website

GitHub Pages deployment:

`https://pkwiatkowski-ssj.github.io/ai-innovation-student-site/`

## Project Purpose

The purpose of this website is to present a polished student innovation platform that highlights:

* AI-focused student projects
* team collaboration
* responsive design
* branding and usability
* portfolio-quality front-end development

The site is being continuously improved through iterative UI updates and structured Git branching workflows.
